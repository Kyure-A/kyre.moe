# kyre.moe

Personal website and blog built with Next.js.

## Highlights

- Next.js 15 + React
- Tailwind CSS with custom motion/graphics (GSAP, Three.js, OGL)
- Markdown-based content pipeline

## Dev

```bash
pnpm install
pnpm dev
```

## Content helpers

```bash
pnpm content:org
pnpm content:images
```
