* kyre.moe

Personal website and blog built with Next.js.

** Highlights

- Next.js 15 + React
- Tailwind CSS with custom motion/graphics (GSAP, Three.js, OGL)
- Markdown-based content pipeline

** Dev

#+begin_src bash
pnpm install
pnpm dev
#+end_src

** Content helpers

#+begin_src bash
pnpm content:org
pnpm content:images
#+end_src
